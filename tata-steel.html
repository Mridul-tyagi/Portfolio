Project Overview

TATA Steel's Construction Management Platform is a digital transformation initiative aimed at modernizing how construction projects are managed across India. The platform connects contractors, project managers, suppliers, and site supervisors in a unified ecosystem, providing real-time visibility into project progress, material inventory, and compliance documentation.

As the lead UI/UX designer, I was responsible for creating a scalable design system and designing 150+ responsive screens that work seamlessly across desktop web and mobile devices. The project followed a mobile-first approach while delivering rich, data-dense dashboards for desktop power users.

Key Design Decisions

📱 1. Mobile-First Approach

Research showed site supervisors rarely have desktop access. We prioritized mobile design for field workers while maintaining rich desktop experiences for managers.

❌ Before

Paper-based forms, no mobile access, delayed reporting

✅ After

Mobile app with offline support, instant photo uploads, real-time sync

📊 2. Progressive Disclosure

Project managers needed detailed metrics without being overwhelmed. We implemented progressive disclosure showing high-level KPIs first with drill-down capabilities.

❌ Before

Crowded dashboards with 50+ metrics, hard to find key info

✅ After

Hierarchical information architecture, expandable sections, smart defaults

🌙 3. Dark Mode Support

Field workers often use devices in low-light conditions. Dark mode reduces eye strain and saves battery life on mobile devices.

❌ Before

Only light mode, eye strain during night shifts

✅ After

Automatic dark mode based on time, manual toggle option

Design Process

1

Research & Discovery

Conducted stakeholder interviews with 8 project managers and site supervisors across 3 construction sites. Analyzed existing workflows and pain points. Defined requirements for both desktop power users and mobile field workers.

2

Information Architecture

Developed sitemaps and user flows for different user roles (Admin, Project Manager, Site Supervisor, Supplier). Created responsive navigation patterns that adapt seamlessly between desktop and mobile.

3

Visual Design & Design System

Created a comprehensive design system including color palette (aligned with TATA Steel brand), typography, component library, and responsive grid system for web + mobile breakpoints.

4

Testing & Iteration

Conducted usability testing with 15 users across desktop and mobile devices. Iterated designs based on feedback, improving task completion time by 25%. Final designs achieved 100% WCAG 2.1 AA compliance.
